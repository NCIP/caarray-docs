CHANGE HISTORY
========================

Version 0.3
8/10/2007
[Model revision for review by Analysis Services Best Practices Working Group]
- Put full array design annotation (Features, Reporters, etc.) back into the registered model.
- Put hybridization data classes into the registered model.

Version 0.2
8/9/2007
- Added an association from Publication to Term to model the "type" of publication (main, reference etc.).
- Added attributes to Publication based on MAGE-OM BibliographicReference (uri, publication, publisher, editor, year, volume, issue, pages).
- Changed "Investigation" to "Experiment", and "InvestigationContact" to "ExperimentContact".
- Changed multiplicity of Experiment's "factors" to 0..*.
- Linked Experiment to Hybridization (many-to-many).
- Added unit and value to FactorValue.
- Allowed samples to have 0..* sources and extracts to have 0..* samples.
- Removed LabeledExtract. Extract or Hybridization can have an associated label.
- Extracts can be transformed into other Extracts (labeled or unlabeled). This satisfies ChIP-chip example shown in MAGE-TAB specification too.
- Added amount and unit attributes to Hybridization.
- Added association from RawArrayData to sourceImages (0..*).
- Added a defaultValue for a Parameter.
- Added a "notes" attribute to ProtocolApplication.
- Added an association from Protocol to CaArrayFile to allow pdf description of protocol etc.
- Expanded Characteristics into TermCharacteristics, ValueCharacteristics and well-known characteristics: Organism, OrganismPart, DiseaseState, DiseaseStaging and Age.
- Searched caDSR and made list of potential reusable CDEs. Need to work with caDSR/EVS contact before annotating model.
- Added a TODO item to add "documentation" tags for all classes and "description" tags for all attributes in order to make the model ready for submission to caDSR.
- Added an ArrayGroup element.

Version 0.1
8/3/2007
- Initial draft reviewed internally.
- Based on MAGE-TAB support with additional considerations for MIAME and caArray 1.x migration.
- Result of E1 iteration.
